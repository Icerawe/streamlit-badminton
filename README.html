🏸 เชียงใหม่แบดมินตัน Ranking System

ระบบจัดอันดับและจัดการทีมแบดมินตัน — Chiang Mai Badminton Ranking
Deploy: cnx-bad-ranking.streamlit.app

วิธีรัน (Local)

# ใช้ uv (แนะนำ)
uv run streamlit run app.py

# หรือใช้ pip
pip install -r app_requirement.txt
streamlit run app.py
ต้องมีไฟล์ .streamlit/secrets.toml ก่อนรัน (ดูหัวข้อ Secrets)

ใช้ Supabase Session Pooler (ไม่ใช่ direct connection) เพื่อรองรับ Streamlit Cloud ที่ใช้ IPv6

หน้าต่างๆ

🏸 ข้อมูลทั่วไป / Information

  • สถิติรวม: จำนวนผู้เล่น, จำนวนทีม
  • ภาพรวมทีมพร้อมจำนวนสมาชิก
  • ตารางประเภทการแข่งขัน + บันไดระดับ rank

📊 อันดับ / Ranking

  • แสดงผู้เล่น 3 การ์ดต่อแถว จัดกลุ่มตาม Rank หรือ Team
  • กรองตามทีม
  • 🚩 flag เมื่อ net protest votes ≥ 5

🔍 จับคู่แข่งขัน / Match Finder

  • Mode จับคู่: เลือก 2 คน → แสดง match categories ที่ใช้ได้
  • Mode จับคู่ภายในทีม: เลือกทีม → แสดง pairings ทั้งหมดที่เป็นไปได้

✊ ประท้วง / Protest

  • โหวต +1/-1 ระดับ ได้คนละ 1 ครั้งต่อผู้เล่น (session-based)
  • บังคับใส่เหตุผล + หลักฐานอย่างน้อย 1 อย่าง (ลิงก์ YouTube หรือภาพรางวัล)
  • อัปโหลดรูปภาพเก็บใน Supabase Storage (S3-compatible)

🔧 ติดต่อสอบถาม / Admin

  • ล็อกอินด้วย ADMIN_PASSWORD จาก secrets
  • เพิ่ม/แก้ไข/ลบผู้เล่น
  • อัปโหลด Excel (format: ทีม / ชื่อเล่น / ระดับ)
  • อนุมัติ/ปฏิเสธ pending players
  • จัดการเสียง protest → ปรับ rank หรือล้าง votes

Deploy บน Streamlit Community Cloud

  1. Push โค้ดขึ้น GitHub (branch: master)
  2. ไปที่ https://share.streamlit.io → เลือก repo → app.py → Deploy
  3. ตั้งค่า Secrets ใน App settings → Secrets
